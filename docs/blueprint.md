# **App Name**: SearchKings Market Analyzer

## Core Features:

- Search Parameter Entry: Input fields for business category, location, and search radius.
- Google Places API Integration: Interface with Google Places API to retrieve business listings based on search parameters.
- Search Results Display: Display business name, address, phone number, website, GBP rating, number of reviews in an organized list.
- Google Ads Detection: Check if the business is running Google Ads by using their website and the Ads Transparency Center.

## Style Guidelines:

- A vibrant blue (#29ABE2), suggestive of trust and intelligence, to convey a sense of reliable insights.
- Light blue (#E1F5FE), very lightly desaturated, providing a clean, unobtrusive backdrop that keeps focus on search results and key information.
- A contrasting purple (#9C27B0), is used sparingly to draw attention to calls to action like 'Search' or actionable information within search result items.
- Clean and modern sans-serif fonts to maintain readability and a professional look.
- Simple, intuitive icons to represent different business categories and actions (e.g., a magnifying glass for search, a star for saving to watchlist).
- A grid-based layout with clear, defined sections for search input, results, and potentially a map view.
- Subtle transitions and loading animations to enhance the user experience without being distracting.
